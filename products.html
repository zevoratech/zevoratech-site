In production Manufacturing · Android app

Factory Operations App

An Android application built for a manufacturing client to run daily floor operations from one place — production tracking, sales management, cash handling, and staff attendance via punch-in/punch-out. Used daily by supervisors and staff on site.

In production Healthcare · Workflow automation

Automated Leave Approval System

A leave request system built for a hospital client so doctors can submit advance leave requests through a simple form. Each request is checked automatically against nine internal policy conditions, with instant auto-approval when all are met — replacing a manual back-and-forth with a process that takes seconds.

Live Manufacturing · Company website

Corporate Website

A company website designed and built for a plastics manufacturing client — covering their story, product range, and a clear way for buyers to get in touch. Live and public-facing today.
